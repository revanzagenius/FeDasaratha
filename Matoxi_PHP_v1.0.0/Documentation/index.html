A) PHP Structure - top

This template is fully responsive and created with Codeigniter 4.4 & Bootstrap 5 framework. Template follow the standard html5 structure. It is starts from header aside, and main content. In "main-wrapper" all main content of Template is available. This is based on SCSS. Here is the general structure.

<?php include 'layouts/main.php'; ?>

<head>
  <?php includeFileWithVariables('layouts/title-meta.php', array('title' => 'Starter Page')); ?>
  
  <?php include 'layouts/head-css.php'; ?>
  
</head>
  
<body>
  
  <?php include 'layouts/menu.php'; ?>
  
	<!--start main wrapper-->
	<main class="main-wrapper">
	  <div class="main-content">
		<!--breadcrumb-->
		<?php includeFileWithVariables('layouts/page-title.php', array('title' => 'Pages', 'pagetitle' => 'Starter Page')); ?>
				  
		<h3 class="mb-0">Main Content</h3>
	  </div>
	</main>
	<!--end main wrapper-->
  
	  <!--start overlay-->
	  <div class="overlay btn-toggle"></div>
	  <!--end overlay-->
  
	 
  <?php include 'layouts/footer.php'; ?>
  
  <?php include 'layouts/cart.php'; ?>
  
  <?php include 'layouts/right-sidebar.php'; ?>
  
  <?php include 'layouts/vendor-scripts.php'; ?>
  
  </body>
  
  </html>

B) Getting Started - top

Prerequisites:

Please follow the below steps to install and setup all prerequisites:

  • Xampp

    Make sure to have the latest version of xampp installed & running on your computer. If you already have installed Xampp on your computer, you can skip this step.

  • PHP 7.4

    Make sure to have PHP-7.4 or latest version installed & running on your computer. If you already have installed PHP on your computer, you can skip this step.

Description
* Set extract zip to in the file location in c:/xammp/htdots/filename
* Create a new database with name common_php, Then import the common_php.sql file in your database.
* Configure the Database credentials in the config.php file.
* You can test locally with signup as a new user to check all the pages.

Email Settings

You can configure the email credentials in the config.php file.
NOTE: Manage your google account security with create app password using below link and set it into config file.
https://myaccount.google.com/apppasswords.
