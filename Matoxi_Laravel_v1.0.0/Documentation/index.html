A) Laravel Structure - ^

This template is fully responsive and created with Laravel 11 & Bootstrap 5 framework. Template follow the standard html5 structure. It is starts from header aside, and main content. In "main-wrapper" all main content of Template is available. This is based on SCSS. Here is the general structure.

<!doctype html>
<html lang="en" data-bs-theme="light">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <!--favicon-->
    <link rel="icon" href="{{ URL::asset('build/images/favicon-32x32.png') }}" type="image/png">
    <title>@yield('title') | Laravel & Bootstrap 5 Admin Dashboard Template</title>

    @yield('css')

    @include('layouts.head-css')

</head>

<body>

@include('layouts.topbar')
@include('layouts.sidebar')

<!--start main wrapper-->
<main class="main-wrapper">
    <div class="main-content">

        @yield('content')

    </div>
</main>
<!--end main wrapper-->

<!--start overlay-->
    <div class="overlay btn-toggle"></div>
<!--end overlay-->

  @include('layouts.footer')

  @include('layouts.cart')

  @include('layouts.right-sidebar')

  @include('layouts.vendor-scripts')

  @yield('scripts')

</body>
  
</html>

B) Getting Started - ^

Installation Steps

Follow these simple steps to install and set up Matoxi - Laravel 11 Admin Template:

  • Step 1: Unzip
    Unzip The Downloaded Folder and open it in vscode or any other code editor you use.

  • Step 2: This would install all of the framework's required dependencies in the vendor folder. 

     composer install

  • Step 3: Please run the below command to generate the new key. 

     php artisan key:generate

  • Step 4: Check public file if storage folder present then delete it. After run the below command to link storage with application. 

     php artisan storage:link

  • Step 5: Please fill your DB credentials in the .env file. 

     DB_CONNECTION=mysql
 DB_HOST=127.0.0.1
 DB_PORT=3306
 DB_DATABASE=matoxi_laravel
 DB_USERNAME=root
 DB_PASSWORD=

  • Step 6: Please fill your email credentials in the .env file. 

     MAIL_MAILER=
 MAIL_HOST=
 MAIL_PORT=
 MAIL_USERNAME=
 MAIL_PASSWORD=
 MAIL_ENCRYPTION=
 MAIL_FROM_ADDRESS="hello@example.com"
 MAIL_FROM_NAME="${APP_NAME}"

  • Step 7: Please run the following commands to clear all cache from the project. 

     php artisan optimize:clear

  • Step 8: This will migrate the database tables. For more details visit Click Here. 

     php artisan migrate

  • Step 9: This will seed the database tables. For more details visit Click Here 

     php artisan db:seed

  • Step 10: The development server is accessible at http://localhost:8000To run on other port just run command : php artisan serve --port=8001 

     php artisan serve  
     php artisan serve --port=8001
